LemonDB Test Framework, by [tc-imba](https://github.com/tc-imba).

```bash

```

```bash
python3 test.py --generate-answer --times=1
```

```bash
python3 test.py -p <project-dir> --times=10
```

```bash
python3 report.py -p <project-dir> -t <group-number>
```
